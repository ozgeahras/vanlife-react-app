## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/vanlife-react-app.git

```

Go to the project directory

```bash
  cd ..vanlife-react-app
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
